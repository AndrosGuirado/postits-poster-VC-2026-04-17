Postits is a SvelteKit 2 / Svelte 5 project using Three.js (`three@^0.184`), GSAP, Tweakpane (debug UI), Pug templates, and Stylus styles. TypeScript 6, Vite 8, pnpm workspace.

**Why:** Stack defines the idiomatic patterns to use (e.g. Svelte 5 runes, not legacy stores; GSAP timelines for animation; Tweakpane for live tuning).

**How to apply:** When suggesting code, default to Svelte 5 runes syntax (`$state`, `$derived`, `$effect`), Pug for templates, Stylus for styles, and Three.js + GSAP idioms. Use pnpm for any install/script commands.
